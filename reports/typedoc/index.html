scrolls-and-decks

Scrolls & Decks ALPHA

Unofficial complementary tool for scrolls.com gamers (and Angular Material Design experiment)

  • Visit the app: darosh.github.io/scrolls-and-decks
  • More information: darosh.github.io/scrolls-and-decks/#/help

It is alpha version and Chrome is the only recommended browser.

Install and build

npm install tested with node.js 0.12

Development

Build and test toolchain

  • gulp check and build, powered by gulp.js
  • gulp serve check, build, serve and watch
  • gulp test check, e2e and unit, output in ./reports
  • gulp e2e e2e test
  • gulp unit unit test (alternative: npm run karma start)
  • gulp chart dependency charts, requires Graphviz, output in ./reports/chart
  • gulp typedoc typedoc output in ./reports/typedoc
  • gulp dry duplication check, output in ./reports/dry
  • gulp plato complexity visualization, output in ./reports/plato
  • gulp clean
  • gulp deploy publish to GitHub pages

Optional gulp parameters

  • --nomin skip minification
  • --norev skip revision hash
  • --nobuild skip build
  • --coverage e2e test coverage

Additional testing

  • .idea project files for WebStorm JavaScript and HTML inspections, run on client/app and client/styles folder
  • custom dictionaries for spell check

Style guides

  • johnpapa/angular-styleguide for AngularJS
  • Material design via angular/material

Documentation

  • ScrollsTypes in TypeScript
